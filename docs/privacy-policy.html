Privacy Policy

Last updated: February 9, 2026

1. Introduction

HomeDocManager ("the Application") is a personal household document management system that automatically organizes, categorizes, and manages documents stored in Google Drive. This privacy policy explains how the Application collects, uses, and protects your data.

The Application is developed and operated by an individual developer for personal and family use. It is not a commercial product and is not intended for public distribution.

2. Data We Access

The Application accesses the following Google services through OAuth 2.0 authentication:

Google Service Permission (Scope) Purpose
Google Drive drive.file (files created by this app only) Create and manage documents generated by the Application for information consolidation
Google Docs documents Write extracted text and summaries to consolidated documents
Google Calendar calendar.events Create calendar events from dates extracted from documents (e.g., school events, appointments)
Google Tasks tasks Create tasks from deadlines extracted from documents (e.g., submission deadlines)
Google Photos photoslibrary.appendonly (upload only) Back up photo-category documents to Google Photos

3. How We Use Your Data

The Application processes your data in the following ways:

  • Document Analysis: Documents placed in a designated Google Drive folder ("Inbox") are analyzed using Google's Gemini AI to determine their category, extract key dates, and perform OCR text extraction.
  • Automatic Organization: Based on the analysis, documents are renamed with a standardized format and moved to appropriate category folders within your Google Drive.
  • Information Extraction: Key dates and deadlines are extracted and created as Google Calendar events or Google Tasks entries.
  • Text Consolidation: OCR text and summaries are appended to consolidated Google Docs documents for reference purposes.
  • Photo Backup: Documents categorized as photos are uploaded to Google Photos for backup.

4. Data Storage and Retention

  • No external storage: The Application does NOT store your document content, personal information, or file data on any external servers or databases.
  • Processing only: Document content is processed in-memory during the analysis phase and is not retained after processing is complete.
  • Your Google Account: All organized files, calendar events, tasks, and consolidated documents remain within your own Google Account.
  • Authentication tokens: OAuth refresh tokens are securely stored in Google Cloud Secret Manager and are used solely for authenticating API requests on your behalf.

5. Data Sharing

The Application does NOT share, sell, rent, or disclose your personal data or document content to any third parties. The only external service used for document analysis is Google's Gemini AI, which processes your documents under Google's own privacy terms.

6. Security Measures

The Application implements the following security measures:

  • All API communications use HTTPS encryption
  • Authentication tokens are stored in Google Cloud Secret Manager (encrypted at rest)
  • Administrative endpoints are protected with token-based authentication
  • The Application runs on Google Cloud Run with Google-managed security infrastructure
  • OAuth permissions are limited to the minimum scopes required for functionality
  • The Application does NOT request full Google Drive access; only app-created files are accessible via the drive.file scope

7. Your Rights

You have the following rights regarding your data:

  • Revoke access: You can revoke the Application's access at any time through your Google Account permissions page.
  • Delete data: Since all data remains in your Google Account, you can delete any files, calendar events, tasks, or documents created by the Application at any time.
  • Request information: You can contact us to request information about what data the Application has accessed.

8. Children's Privacy

The Application is designed for use by adults managing household documents. It is not directed at children under the age of 13. The Application does not knowingly collect personal information from children.
